A complete list of commonly used Linux commands from **A to Z**.

### **A**

* **alias** – Create shortcuts for commands
* **apt** – Package manager for Debian/Ubuntu
* **awk** – Text processing tool

### **B**

* **basename** – Strip directory and suffix from filenames
* **bash** – GNU shell
* **bg** – Resume job in background

### **C**

* **cat** – View file content
* **cd** – Change directory
* **chmod** – Change file permissions
* **chown** – Change file owner
* **cp** – Copy files
* **cron** – Job scheduler
* **curl** – Transfer data to/from server

### **D**

* **date** – Show system date/time
* **df** – Disk space usage
* **diff** – Compare files
* **du** – Directory size usage

### **E**

* **echo** – Print text
* **env** – Show environment variables
* **exit** – Exit shell

### **F**

* **find** – Search for files
* **free** – Show memory usage
* **fg** – Bring job to foreground

### **G**

* **grep** – Search text inside files
* **groupadd** – Create a group
* **git** – Version control system

### **H**

* **head** – First lines of file
* **hostname** – Show hostname
* **htop** – Interactive process viewer

### **I**

* **id** – Show user identity
* **ifconfig** – Network info
* **ip** – Modern network management

### **J**

* **jobs** – Show active jobs

### **K**

* **kill** – Stop a process by PID
* **kubectl** – Kubernetes command-line tool

### **L**

* **ls** – List files
* **less** – Scroll file output
* **ln** – Create links
* **logrotate** – Manage log files

### **M**

* **man** – Manual pages
* **mkdir** – Create directory
* **mv** – Move/rename files
* **mount** – Mount storage

### **N**

* **nano** – Terminal editor
* **netstat** – Network statistics
* **nslookup** – DNS query tool

### **O**

* **openssl** – Toolkit for SSL/TLS
* **passwd** – Change user password

### **P**

* **ping** – Test network
* **ps** – Show processes
* **pwd** – Current directory

### **Q**

* **quota** – Disk quota information

### **R**

* **rm** – Delete files
* **rmdir** – Remove empty directory
* **rsync** – File sync over network
* **reboot** – Restart system

### **S**

* **ssh** – Remote login
* **scp** – Copy files over SSH
* **sed** – Stream editor
* **systemctl** – Manage services

### **T**

* **tail** – Last lines of file
* **tar** – Archive files
* **top** – Process viewer
* **touch** – Create empty files

### **U**

* **uname** – System information
* **uptime** – System load & uptime
* **useradd** – Add user
* **unlink** – Remove file

### **V**

* **vim** – Advanced text editor
* **version** – Show version (varies per tool)

### **W**

* **who** – Show logged-in users
* **whoami** – Current user
* **wget** – Download files

### **X**

* **xdg-open** – Open a file or URL (Linux GUI)

### **Y**

* **yum** – Package manager for RHEL/CentOS

### **Z**

* **zip** – Compress files
* **zcat** – View compressed files
* **zgrep** – Grep inside gzip files
